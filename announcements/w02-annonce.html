Bonjour à tous !

Le MOOC Python 3 commence sa deuxième semaine. Au programme cette semaine, nous commençons à étudier les types de base du langage, chaines de caractères et listes notamment; et puis nous survolerons quelques constructions de base qui nous permettrons de commencer à écrire du vrai code - fonctions, boucles for et tests, notamment.

Grâce à la contribution de Fangh, vous trouverez en ligne les supports de cours au format pdf, voyez sur le côté droit de la présente page "Infos du cours", sous le logo.

Comme toujours, n'hésitez pas à utiliser le forum de discussion pour partager vos questionnements et trouvailles. Vous y trouverez notamment:

  • des astuces pour mettre en forme votre code dans le forum,
  • et des pistes pour régler votre navigateur si vous rencontrez des difficultés techniques, que ce soit pour faire fonctionner la navigation basée sur les sous-titres dans les vidéos, ou pour utiliser les notebooks.

Enfin, nous n'avons jamais trouvé "la bonne place" pour la section sur les expressions régulières. Nous l'avons mise cette fois très tôt dans le cours (séquence #2 sur les chaines de caractères) mais elle pourra paraître indigeste aux débutants. N'hésitez surtout pas à la laisser de côté, pour y revenir plus tard le cas échéant.

Bonne deuxième semaine !
